---
title: "Create Rule"
api: "POST /api/v1/gateway/{name}/moderation/rules"
description: "Create a new moderation rule"
---

## Path Parameters

<ParamField path="name" type="string" required>Gateway name.</ParamField>

## Request Body

<ParamField body="description" type="string" required>
  Natural language description of the behavior to moderate.
</ParamField>
<ParamField body="actionType" type="string" required>
  Action to take: `mute`, `kick`, or `ban`.
</ParamField>
<ParamField body="muteDuration" type="number">
  Duration in minutes (only for `mute` action, 0 for permanent).
</ParamField>
<ParamField body="permanent" type="boolean">
  Whether the action is permanent.
</ParamField>
<ParamField body="enabled" type="boolean">
  Whether the rule is active. Defaults to `true`.
</ParamField>

## Response

Returns the created rule object.

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "id": "rule-abc123",
    "description": "Users posting spam or advertising links",
    "actionType": "mute",
    "muteDuration": 30,
    "permanent": false,
    "enabled": true
  }
}
```
