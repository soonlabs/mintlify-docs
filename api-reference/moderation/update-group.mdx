---
title: "Update Group"
api: "PUT /api/v1/gateway/{name}/groups/{id}"
description: "Update group settings"
---

## Path Parameters

<ParamField path="name" type="string" required>Gateway name.</ParamField>
<ParamField path="id" type="number" required>Group ID (internal).</ParamField>

## Request Body

<ParamField body="requireMention" type="boolean">
  Whether the bot requires an @mention to respond.
</ParamField>
<ParamField body="fallbackLanguage" type="string">
  Fallback language code. Pass `null` for auto-detect.
</ParamField>

## Response

Returns the updated group object.

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "id": 1,
    "chatId": "-1001234567890",
    "title": "My Community",
    "requireMention": true,
    "fallbackLanguage": "en"
  }
}
```

<Note>The gateway must be online to update group settings.</Note>
