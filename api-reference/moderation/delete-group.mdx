---
title: "Delete Group"
api: "DELETE /api/v1/gateway/{name}/groups/{id}"
description: "Remove a Telegram group from a gateway"
---

## Path Parameters

<ParamField path="name" type="string" required>Gateway name.</ParamField>
<ParamField path="id" type="number" required>Group ID (internal).</ParamField>

## Response

```json
{
  "code": 0,
  "message": "OK",
  "data": null
}
```

<Note>This only disconnects the group from the gateway. The bot remains a member of the Telegram group — remove it manually if needed.</Note>
