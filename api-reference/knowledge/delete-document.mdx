---
title: "Delete Document"
api: "DELETE /api/v1/gateway/{name}/knowledge/{id}"
description: "Delete a knowledge base document"
---

## Path Parameters

<ParamField path="name" type="string" required>Gateway name.</ParamField>
<ParamField path="id" type="string" required>Document ID to delete.</ParamField>

## Response

```json
{
  "code": 0,
  "message": "OK",
  "data": null
}
```

<Warning>This action is irreversible. The document and its indexed data will be permanently removed.</Warning>
