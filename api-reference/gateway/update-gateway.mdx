---
title: "Update Gateway"
api: "PUT /api/v1/gateway/{name}"
description: "Update gateway configuration"
---

## Path Parameters

<ParamField path="name" type="string" required>
  Gateway name.
</ParamField>

## Request Body

<ParamField body="model" type="string">
  AI model. Options: `gpt-4o`, `gpt-4o-mini`.
</ParamField>

<ParamField body="telegramToken" type="string">
  Telegram bot token.
</ParamField>

<ParamField body="persona" type="object">
  Personality configuration.
  <Expandable title="Persona fields">
    <ParamField body="preset" type="string">Preset name: `friendly`, `professional`, `creative`, `concise`, `custom`.</ParamField>
    <ParamField body="customDescription" type="string">Custom personality description (used when preset is `custom`).</ParamField>
  </Expandable>
</ParamField>

## Response

Returns the updated gateway object.

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "name": "my-agent",
    "model": "gpt-4o",
    "persona": {
      "preset": "custom",
      "customDescription": "You are a helpful Web3 community manager..."
    }
  }
}
```
