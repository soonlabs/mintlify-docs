---
title: "Get Gateway"
api: "GET /api/v1/gateway/{name}"
description: "Get detailed information about a specific gateway"
---

## Path Parameters

<ParamField path="name" type="string" required>
  Gateway name.
</ParamField>

## Response

Returns the full gateway detail including status, configuration, and persona settings.

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "name": "my-agent",
    "slotId": "slot-abc123",
    "status": {
      "ready": true,
      "suspended": false
    },
    "persona": {
      "preset": "friendly",
      "customDescription": "",
      "avatarId": 1
    },
    "model": "gpt-4o",
    "telegramToken": "123456:ABC..."
  }
}
```
