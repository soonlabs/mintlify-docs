---
title: "Create Gateway"
api: "POST /api/v1/gateway"
description: "Create a new gateway (Bot)"
---

Creates a new Bot. The platform assigns a pre-registered Telegram Bot from its internal Bot Pool — **no Telegram token is required** in the request.

## Request Body

<ParamField body="name" type="string" required>
  Bot slug — lowercase letters, numbers, and hyphens only (max 63 chars). **Cannot be changed after creation.**
</ParamField>

<ParamField body="selectedModel" type="string" required>
  AI model ID. One of:
  - **Economy**: `gpt-4.1-nano`, `gpt-4o-mini`
  - **Balanced**: `gpt-5-mini` (recommended default), `gpt-4.1-mini`
  - **Premium**: `gpt-5.2`, `gpt-4o`, `gpt-4.1`
</ParamField>

<ParamField body="presetId" type="string">
  Community preset ID. One of: `project-community`, `kol-alpha`, `trader-community`, `meme-community`. Omit or use `custom` to skip preset and configure persona manually.
</ParamField>

## Response

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "name": "my-project-bot",
    "namespace": "ns-...",
    "slotId": "slot-abc123",
    "botUsername": "my_assigned_bot"
  }
}
```

- `botUsername` is the Telegram handle assigned from the Bot Pool — you can open it directly at `https://t.me/<botUsername>`.

<Note>The gateway takes 20–30 seconds to provision. Poll [List Gateways](/api-reference/gateway/list-gateways) to check when `status.ready` becomes `true`.</Note>

<Warning>Requires an active Bot purchase. Returns a billing error otherwise.</Warning>
