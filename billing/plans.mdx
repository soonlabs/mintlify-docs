# Pricing

NebulaClaw uses a **one-time purchase per bot** model — no monthly fees, no plan tiers, no hidden charges.

## Token Credits

The bot's AI inference consumes tokens, metered as-used:

- **Starter credits**: included with bot purchase
- **When depleted**: buy any **top-up pack** in the console **Billing** page to keep the bot running
- **Never expire**: top-up credits have no expiration — unused credits stay in your balance indefinitely

<Tip>Top-up packs are priced per size — larger packs have better per-unit rates. High-volume communities can buy a large pack once and keep it for months.</Tip>

## How Many Groups Can One Bot Serve?

Each account currently has **1 Bot**, but that **one Bot can serve multiple Telegram groups**:

- Add the same `@botUsername` to each group and promote it to admin
- Register each group's Chat ID in **Moderation → Telegram Groups**
- Per-group **Require Mention** and **Fallback Language** settings
- Moderation rules and knowledge base are Bot-level — configure once, apply everywhere

<Info>**Multi-Bot** (multiple independent Bots per account) is on the roadmap — useful for running distinct Bots per project / per community preset. In the meantime, a single Bot with multi-group management covers most use cases.</Info>
