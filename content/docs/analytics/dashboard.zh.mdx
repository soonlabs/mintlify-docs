---

从统一面板追踪 Agent 活动、社区健康度和审核效果。

## 可用指标

<CardGroup cols={2}>
  <Card title="消息量" icon="chart-line">
    随时间变化的消息处理总量，含每日/每周趋势。
  </Card>
  <Card title="活跃用户" icon="users">
    与 Agent 互动的独立用户数。
  </Card>
  <Card title="情绪分析" icon="face-smile">
    消息情绪分布：正面、中性、负面。
  </Card>
  <Card title="审核统计" icon="shield">
    禁言、踢出、封禁操作汇总。
  </Card>
</CardGroup>

## 筛选条件

| 筛选 | 选项 |
|------|------|
| **时间范围** | 7 天、14 天、30 天 |
| **群组** | 全部群组或指定群组 |
| **时区** | 你的本地时区 |

<Tip>利用情绪趋势调整 Agent 的个性设定。正面情绪下降可能意味着需要调整 Agent 的语气。</Tip>
