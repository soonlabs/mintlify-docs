## Bot（Agent）

**Bot** 是你的 AI 实例，一个 Bot 对应一条独立的 Agent 网关：自己的 AI 模型、人设（Persona）、知识库、审核规则和连接的 Telegram 群组。

购买 Bot 后平台会从内置的 **Bot Pool** 自动分配一个已注册的 Telegram Bot 账号并托管运行——**你不需要和 @BotFather 打交道**。网关启动约 30 秒，控制台自动轮询状态。

<Warning>**Bot 的头像和用户名创建后暂不支持修改。** 命名请谨慎。</Warning>

## 社群定位预设

NebulaClaw 预置了四种**社群定位**，每种预设内置经过调优的 Persona Prompt、头像和话术风格：

| 预设 | 适合社群 |
|------|---------|
| **项目社群** | Web3 项目方 / 产品社区——白皮书答疑 + 反 shill |
| **KOL / Alpha 群** | 付费 Alpha / 研究群——信噪比控制、DYOR 提醒 |
| **交易员社群** | 交易讨论群——市场中立、不提供投资建议 |
| **Meme 社群** | Degen / Meme 币群——高能量氛围 + 严查诈骗链接 |

选定预设后，可以进一步进入 **自定义** 追加项目专属指令（例如"遇到质押问题引导到 /staking"）。

<Tip>切换预设时会保留你的自定义文本，可以放心尝试。</Tip>

## AI 模型档位

三档分级，覆盖不同成本 / 质量需求：

| 档位 | 示例模型 | 适合 |
|------|---------|------|
| **Economy** | `gpt-4.1-nano`、`gpt-4o-mini` | 大消息量、成本敏感 |
| **Balanced**（默认推荐） | `gpt-5-mini`、`gpt-4.1-mini` | 通用场景，质量与成本平衡 |
| **Premium** | `gpt-5.2`、`gpt-4o`、`gpt-4.1` | 高要求问答、复杂审核 |

模型可在创建后随时在 Agent 设置里切换。

## 知识库

**知识库** 是 Agent 回答问题时参考的文档集合。每个 Bot 最多 **50 个文档**，单文档正文上限 **50 KB**（约 2.5 万中文字），标题上限 256 字节。

文档会被**语义索引**——即使用户没用精确关键词，Agent 也能找到相关段落。

## 审核规则

**审核规则** 用自然语言描述，由 AI 理解并执行，不需要维护关键词黑名单：

> "发布带'保底翻倍'、'官方空投'话术的诱导消息"

可用动作：

| 动作 | 说明 |
|------|------|
| **mute** | 暂时限制发言，可设分钟数或永久 |
| **kick** | 移出群组，可通过邀请链接重新加入 |
| **ban** | 永久移出群组，需手动解封 |

所有动作进入 **Audit 日志**，包含触发规则、上下文、时间戳，方便合规回溯。

## 渠道

**渠道** 是 Bot 连接的消息平台：

- **Telegram** — 完整支持，群组管理、@ 提及控制、多语言回退
- **Discord** — 规划中
- **Slack** — 规划中

## 群组与回退语言

每个 Bot 可服务多个 Telegram 群组。群组需**手动添加 Chat ID**（`-100...`）到 **Moderation → Telegram Groups**。每个群组可独立配置：

- **Require Mention（需要 @）**：开启后 Bot 只在被 @ 时回复；关闭则监听全部消息。
- **Fallback Language（回退语言）**：Bot 默认自动识别用户语种并用对应语言回复。**当无法识别时**（例如用户只发了表情、图片或混合语言），Bot 会用这个设定的回退语言回复。默认"**跟随用户**"（auto），也可指定为固定语言（英 / 中 / 韩 / 土 / 印尼 / 俄 / 越）。

## Token 额度

Bot 的 AI 推理按 Token 计量：

- **月度额度**：$9.99/月订阅自带 $5.00 Token 额度，每个计费周期结束时**自动清零**
- **充值包额度**：在 Billing 按需购买，**永不过期**
- 月度额度优先消耗，用完后自动切换充值包余额；两者都耗尽时 Bot 暂停，充值后立即恢复

详见 [定价](/docs/billing/plans)。
