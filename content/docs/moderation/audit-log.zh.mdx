---
title: 审核日志
description: "查看自动审核操作记录"
---

审核日志记录 Agent 执行的每一次审核操作，提供完整的自动化执行透明度。

## 查看日志

进入 **审核管理** 页面，点击 **审核日志** 标签。每条记录包含：

| 列 | 说明 |
|----|------|
| **时间** | 操作执行时间 |
| **目标** | 被操作用户的用户名或 ID |
| **操作** | 执行的操作类型（禁言、踢出、封禁）|
| **原因** | 触发操作的审核规则 |
| **群组** | 操作发生的群组 |
| **状态** | 执行结果 |

## 操作状态

| 状态 | 含义 |
|------|------|
| **成功** | 操作执行成功 |
| **等待** | 操作排队等待执行 |
| **重试中** | 执行失败，自动重试 |
| **失败** | 执行失败（悬停查看错误详情）|
| **永久失败** | 所有重试均失败 |

## 分页

每页显示 20 条记录，使用 **上一页** / **下一页** 按钮翻页。左下角显示总条数。

<Tip>定期查看审核日志检查误判。如果某条规则频繁对正常消息触发，请修改规则描述使其更具体。</Tip>
