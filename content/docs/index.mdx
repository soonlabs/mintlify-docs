**NebulaClaw is an AI-powered community automation platform.** It replaces rigid `/command`-and-keyword bots with a context-aware AI Agent that understands natural language.

The three core community workloads — **answering questions, moderating messages, and surfacing operational insight** — are handled by the Agent automatically. You describe what you want in plain English; deployment takes about five minutes, no code required.

## Technical Highlights

- **Auto-provisioned Bot** — after checkout, the platform creates and hosts your Telegram Bot automatically. No BotFather interaction required.
- **Semantic Moderation** — rules are written in natural language (e.g. "users posting promotional external links"); the AI judges by context, dramatically reducing false positives compared to keyword matching.
- **Auto Language Detection** — detects the user's language and replies in kind; fallback language configurable across English, Chinese, Korean, Turkish, Indonesian, Russian, and Vietnamese.
- **$9.99/month subscription** — includes $5 monthly token credits (reset at the end of each billing cycle); buy top-up packs for extra usage — **top-up credits never expire**.

## Pain Points We Solve

From a project / operations team perspective, NebulaClaw addresses six key dimensions:

<CardGroup cols={2}>
  <Card title="Staffing Cost" icon="user-minus">
    **Pain:** 24/7 community coverage needs three-shift community managers — multiplied by languages for Web3 / cross-border projects. Holidays and late nights are exactly when scam links thrive.

    **Solution:** One Agent replaces the coverage layer — auto-reply, auto-moderate, never offline. Your team only handles what the Agent escalates.
  </Card>
  <Card title="Response Time" icon="timer">
    **Pain:** A single airdrop-phishing / fake-cashback message can complete the con in under 30 seconds; human mods respond in minutes — by then the damage is done.

    **Solution:** The Agent detects and acts in **seconds**, closing the window before phishing spreads.
  </Card>
  <Card title="Rule Maintainability" icon="wrench">
    **Pain:** Keyword / regex dictionaries need constant updates as bad actors rotate domains and scripts. Miss one and trust erodes; over-block and users leave.

    **Solution:** Describe intent in **natural language** (e.g. "invite links to non-whitelisted domains", "fake airdrop / referral-farming messages"). The AI judges by context — no dictionaries to maintain; precision scales with clarity of description.
  </Card>
  <Card title="Multilingual Coverage" icon="languages">
    **Pain:** Web3 / cross-border communities naturally mix EN / ZH / KO / TR / ID / RU / VI. Splitting groups is costly; merging leads to "misjudged because unreadable."

    **Solution:** **Auto-detects 7 languages**, replies in the user's language, enforces rules across all of them — no separate groups or localized dictionaries needed.
  </Card>
  <Card title="Compliance & Audit" icon="scroll-text">
    **Pain:** Human mods apply rules inconsistently; follow-up appeals or disputes lack an evidence chain. DAOs and listed companies are especially exposed.

    **Solution:** Every action goes into the **Audit Log** — triggered rule, context, action type, timestamp. Uniform enforcement, fully traceable — ready for internal and compliance review.
  </Card>
  <Card title="Operational Insight" icon="chart-column">
    **Pain:** Traditional bots take actions but leave no data. Leadership can't see community health; KOL partnerships and campaign retros run on gut feel.

    **Solution:** Built-in **analytics dashboard** — message volume, active users, sentiment trends, quantified moderation actions. Turns the "vibes black box" into a measurable asset.
  </Card>
</CardGroup>

<Card title="Bonus · Deployment & Scale" icon="rocket">
