---
title: "Delete Gateway"
api: "DELETE /api/v1/gateway/{name}"
description: "Delete a gateway and all associated data"
---
## Path Parameters
<ParamField path="name" type="string" required>
  Gateway name to delete.
</ParamField>
## Response
```json
{
  "code": 0,
  "message": "OK",
  "data": null
}
```
<Warning>This action is irreversible. All moderation rules, knowledge documents, and group configurations associated with this gateway will be permanently deleted.</Warning>
