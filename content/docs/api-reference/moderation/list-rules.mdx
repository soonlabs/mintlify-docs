---
title: "List Rules"
api: "GET /api/v1/gateway/{name}/moderation/rules"
description: "List all moderation rules for a gateway"
---
## Path Parameters
<ParamField path="name" type="string" required>Gateway name.</ParamField>
## Response
<ResponseField name="items" type="array">
  Array of moderation rule objects.
  <Expandable title="Rule fields">
    <ResponseField name="id" type="string">Rule ID</ResponseField>
    <ResponseField name="description" type="string">Natural language rule description</ResponseField>
    <ResponseField name="actionType" type="string">Action: `mute`, `kick`, or `ban`</ResponseField>
    <ResponseField name="muteDuration" type="number">Mute duration in minutes (0 if not applicable)</ResponseField>
    <ResponseField name="permanent" type="boolean">Whether the action is permanent</ResponseField>
    <ResponseField name="enabled" type="boolean">Whether the rule is active</ResponseField>
  </Expandable>
</ResponseField>
```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "items": [
      {
        "id": "rule-abc123",
        "description": "Users posting spam or advertising links",
        "actionType": "mute",
        "muteDuration": 30,
        "permanent": false,
        "enabled": true
      }
    ]
  }
}
```
