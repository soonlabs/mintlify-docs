---
title: "Update Rule"
api: "PUT /api/v1/gateway/{name}/moderation/rules/{id}"
description: "Update an existing moderation rule"
---
## Path Parameters
<ParamField path="name" type="string" required>Gateway name.</ParamField>
<ParamField path="id" type="string" required>Rule ID.</ParamField>
## Request Body
<ParamField body="description" type="string">Rule description.</ParamField>
<ParamField body="actionType" type="string">Action: `mute`, `kick`, or `ban`.</ParamField>
<ParamField body="muteDuration" type="number">Mute duration in minutes.</ParamField>
<ParamField body="permanent" type="boolean">Whether the action is permanent.</ParamField>
<ParamField body="enabled" type="boolean">Whether the rule is active.</ParamField>
## Response
Returns the updated rule object.
```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "id": "rule-abc123",
    "description": "Updated rule description",
    "actionType": "kick",
    "muteDuration": 0,
    "permanent": false,
    "enabled": true
  }
}
```
