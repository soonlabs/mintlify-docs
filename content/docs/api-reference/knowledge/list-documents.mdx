---
title: "List Documents"
api: "GET /api/v1/gateway/{name}/knowledge"
description: "List all knowledge base documents for a gateway"
---
## Path Parameters
<ParamField path="name" type="string" required>Gateway name.</ParamField>
## Response
<ResponseField name="items" type="array">
  Array of knowledge document objects.
  <Expandable title="Document fields">
    <ResponseField name="id" type="string">Document ID</ResponseField>
    <ResponseField name="title" type="string">Document title</ResponseField>
    <ResponseField name="content" type="string">Document content</ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 creation timestamp</ResponseField>
    <ResponseField name="updatedAt" type="string">ISO 8601 last update timestamp</ResponseField>
  </Expandable>
</ResponseField>
```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "items": [
      {
        "id": "doc-abc123",
        "title": "FAQ",
        "content": "Q: What is NebulaClaw?\nA: An AI community platform...",
        "createdAt": "2025-01-15T10:30:00Z",
        "updatedAt": "2025-01-15T10:30:00Z"
      }
    ]
  }
}
```
