# Telegram 集成

NebulaClaw 目前支持 **Telegram** 作为消息渠道。Bot 由平台自动托管——**你不需要和 @BotFather 打交道**。

## 前提条件

- 已购买 Bot（$9.9 / Bot）并完成人设、模型配置
- 一个 Telegram 账号和要接入的群组

## 把 Bot 加入群组

<Steps>
  <Step title="获取 Bot 用户名">
    在控制台 Bot 概览页（Agent 详情）可以看到 Bot 的 `@botUsername`，也可以点 **在 Telegram 中打开** 直接跳转私聊。
  </Step>

  <Step title="邀请加入群">
    打开 Telegram 群组 → 添加成员 → 搜索 `@botUsername` 加入。
  </Step>

  <Step title="设为管理员">
    把 Bot 提升为 **群管理员**。这是审核功能（mute / kick / ban）生效的必要权限。

    <Warning>如果 Bot 只是普通成员，它能回复消息但**无法执行审核动作**。</Warning>
  </Step>

  <Step title="在控制台添加群组">
    进入 **Moderation → Telegram Groups**，在输入框粘贴群组 Chat ID（`-100` 开头），点击 **添加群组**。
  </Step>
</Steps>

## 获取 Chat ID

打开 [@userinfobot](https://t.me/userinfobot) 的**私聊**（不需要加它进群）：

1. 点击底部 **Menu** 按钮
2. 选择 **Group**（所有你管理的群）或 **My Group**（你自己创建的群）
3. 从弹出的列表中选择目标群组
4. Bot 立即返回该群的 Chat ID（格式 `-1001234567890`）

<Tip>超级群组（Supergroup）的 Chat ID 一定以 `-100` 开头，这是 Telegram 协议规范。同理可用 **Channel / Forum** 选项获取频道或 Forum 的 ID。</Tip>

## 群组设置

在 **Moderation → Telegram Groups** 每个群组可单独配置：

### Require Mention（需要 @）

| 设置 | 行为 |
|------|------|
| **开启** | Bot 只在被 `@botUsername` 提及时回复 |
| **关闭** | Bot 监听所有消息，满足条件时主动回复 |

> 对于低消息量的精品群，关闭"需要 @"让 Bot 显得更主动；高消息量群建议开启以节省 Token。

### Fallback Language（Bot 回复语言）

控制 Bot 在该群的回复语言：

- **跟随用户**（默认，"auto"）—— Bot 自动识别每条消息的语种并用对应语言回复
- 指定固定语言——Bot 始终用该语言回复，忽略用户语言：

  > 英语、简体中文、韩语、土耳其语、印尼语、俄语、越南语

**场景举例**：

- 中文项目方的中文群 → 设 **简体中文**，避免 Bot 在少数英文消息下切换成英文回复
- 跨境多语社区 → 保持 **跟随用户**，Bot 自动匹配每个人的语言
- 面向海外的官方英文群 → 设 **英语**，统一输出语调

<Note>Bot 在线时才能修改群组设置。离线时设置为只读。</Note>

## 常见问题

<AccordionGroup>
  <Accordion title="Bot 在群组里完全不回复">
    依次排查：① Bot 已加群且设为群管理员；② 群组 Chat ID 已在 **Moderation → Telegram Groups** 添加；③ 如果开启了"需要 @"，直接 @ Bot 测试；④ 控制台 Bot 状态显示为在线（绿色）。
  </Accordion>
  <Accordion title="Bot 能回复但无法禁言 / 踢人">
    Bot 必须是**群管理员**才能执行审核动作。把它从普通成员提升为 Admin 即可。
  </Accordion>
  <Accordion title="Bot 回复的语言不对">
    进入 **Moderation → Telegram Groups**，把该群的 **Fallback Language** 改成你想要的固定语言即可。如果希望 Bot 按用户语言自动切换，设为 **跟随用户**（默认）。
  </Accordion>
  <Accordion title="可以一个 Bot 管多个群吗？">
    可以。同一个 Bot 可以被加入多个群组，每个群在 **Moderation → Telegram Groups** 独立配置。
  </Accordion>
</AccordionGroup>
