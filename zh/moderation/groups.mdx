# 群组管理

Telegram 群组表格显示 Agent 连接的所有群组及其配置选项。

## 群组列表

| 列 | 说明 |
|----|------|
| **群组** | 群组名称（来自 Telegram）|
| **Chat ID** | Telegram 群组标识符 |
| **需要 @** | Bot 是否需要被 @ 提及才回复 |
| **回退语言** | 无法检测用户语言时的默认语言 |
| **操作** | 删除按钮 |

## 添加群组

群组会在你将 Bot 添加到 Telegram 群组时 **自动检测**。也可以手动添加：

1. 输入 Telegram Chat ID（如 `-1001234567890`）
2. 选择回退语言
3. 点击 **添加群组**

<Tip>获取 Chat ID：将 [@userinfobot](https://t.me/userinfobot) 添加到群组，它会自动显示 Chat ID。</Tip>

## 群组设置

### 需要 @ 提及

| 设置 | 行为 |
|------|------|
| **开启** | Bot 只在被 @ 提及时回复 |
| **关闭** | Bot 监控所有消息，可能主动回复 |

### 回退语言

- **跟随用户**（默认）— 自动检测
- 英语、简体中文、韩语、土耳其语、印尼语、俄语、越南语

<Warning>Agent 必须在线才能修改群组设置。离线时设置为只读。</Warning>
