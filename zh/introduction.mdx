# 欢迎使用 NebulaClaw

NebulaClaw 是一个 AI 驱动的社区自动化平台，用智能 Agent 替代传统的机器人命令。几分钟内即可在 Telegram 群组中部署 AI Agent，无需编写代码。

## NebulaClaw 能做什么

<CardGroup cols={2}>
  <Card title="AI Agent" icon="robot">
    部署理解自然语言的智能 Agent，根据上下文回复社区成员。
  </Card>
  <Card title="智能审核" icon="shield-check">
    用自然语言定义审核规则，AI 自动识别并执行——禁言、踢出或封禁。
  </Card>
  <Card title="知识库" icon="brain">
    上传文档，让 Agent 能回答关于你的项目、产品或社区规则的问题。
  </Card>
  <Card title="数据分析" icon="chart-line">
    实时追踪消息量、活跃用户、情绪趋势和审核操作。
  </Card>
</CardGroup>
