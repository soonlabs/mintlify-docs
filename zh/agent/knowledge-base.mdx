# 知识库

知识库让你上传文档供 AI Agent 参考，将通用 AI 变成你社区的领域专家。

## 工作原理

当社区成员提问时，Agent 会：

1. 在知识库中语义搜索相关段落
2. 将匹配内容作为上下文生成回复
3. 基于你的文档内容准确回答

## 添加文档

<Steps>
  <Step title="进入知识库">
    从侧边栏进入 **知识库** 页面。
  </Step>

  <Step title="创建文档">
    点击 **添加文档**，填写：
    - **标题** — 描述性名称（如"常见问题"、"代币经济学"、"社区规则"）
    - **内容** — 粘贴或输入文档文本
  </Step>

  <Step title="保存">
    点击 **保存**，文档自动索引。
  </Step>
</Steps>

## 最佳实践

<Tip>
**保持文档聚焦。** 每个文档专注一个主题，比一个大文档涵盖所有内容效果更好。
</Tip>

**推荐用途：**
- 常见问题（FAQ）
- 产品/项目文档
- 社区规则和指南
- 技术支持排错指南
- 新成员入门资料

<Note>文档会进行语义索引，即使用户没有使用文档中的精确关键词，Agent 也能找到相关内容。</Note>
