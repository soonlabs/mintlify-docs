# 个性配置

个性定义了 Agent 的语气、风格和行为方式。可以选择预设或完全自定义。

## 预设个性

| 预设 | 描述 |
|------|------|
| **友好** | 随和亲切，使用表情符号，对话风格温暖 |
| **专业** | 正式严谨，结构化回复，适合商务场景 |
| **创意** | 富有想象力，使用比喻、幽默和创意表达 |
| **简洁** | 言简意赅，直击要点，不含多余内容 |

点击任意预设卡片即可应用，右侧聊天预览会立即更新。

## 自定义个性

选择 **自定义** 后可以编写个性描述，自由定义：

- Agent 的说话方式（语气、正式程度、语言风格）
- 需要关注或避免的话题
- 特定指令或限制
- 角色设定

**示例：**
```
你是一个友好的 Web3 社区管理员。用专业但易懂的语气回复。
回答技术问题时始终附上官方文档链接。
用简单的语言解释复杂的区块链概念。
不要给出任何投资建议。
```

<Tip>切换预设时会保留自定义文本，可以放心尝试不同风格再切回自定义。</Tip>

## 保存更改

修改后页面底部会出现 **保存** 按钮，点击即可生效。

<Note>保存后立即生效，无需重启 Agent。</Note>
