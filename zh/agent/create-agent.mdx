# 创建 Agent

<Steps>
  <Step title="登录控制台">
    前往 [nonprod.nebulaclaw.ai](https://nonprod.nebulaclaw.ai)，使用 Google 账号登录。
  </Step>

  <Step title="开始创建">
    如果还没有 Agent，登录后会自动进入创建页面。
  </Step>

  <Step title="配置 Agent">
    - **名称** — 给 Agent 取一个描述性的名称
    - **AI 模型** — 选择模型：

    | 模型 | 适用场景 |
    |------|----------|
    | GPT-4o | 高质量回复，复杂对话 |
    | GPT-4o-mini | 高性价比，高消息量群组 |

    <Info>推荐大多数场景使用 GPT-4o。标有"推荐"徽章的模型在质量和成本之间取得了最佳平衡。</Info>
  </Step>

  <Step title="等待配置完成">
    点击 **创建** 后，Agent 开始配置，通常需要 20-30 秒。控制台会自动轮询状态，无需刷新页面。
  </Step>

  <Step title="Agent 就绪">
    状态指示灯变绿后，Agent 已上线，可以连接渠道了。
  </Step>
</Steps>

<Warning>Agent 名称创建后不可更改，请谨慎选择。</Warning>
