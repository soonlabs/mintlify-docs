# 核心概念

## Agent（网关）

**Agent** 是你的 AI 机器人实例。每个 Agent 拥有独立的 AI 模型、个性设定、知识库和渠道连接。创建 Agent 时，NebulaClaw 会自动配置一个专属网关来处理所有消息和审核。

Agent 配置大约需要 30 秒。控制台会自动轮询启动状态。

## 个性

**个性** 控制 Agent 的沟通方式。四种预设可选：

- **友好** — 随和亲切，使用表情符号
- **专业** — 正式严谨，商务风格
- **创意** — 富有想象力，善用比喻
- **简洁** — 言简意赅，直击要点

你也可以编写 **自定义** 个性 —— 自由描述 Agent 的行为方式、语气和特定指令。

<Tip>切换预设时会保留你的自定义文本，可以放心尝试。</Tip>

## 知识库

**知识库** 是 Agent 回答问题时参考的文档集合。上传 FAQ、产品文档、社区规则等文本内容。

文档会进行语义索引 —— 即使用户没使用精确关键词，Agent 也能找到相关段落。

## 审核规则

**审核规则** 使用自然语言描述，由 AI 理解并执行。无需编写正则表达式或关键词列表：

> "用户发布垃圾广告或推广链接"

AI 能理解上下文和语义，相比关键词匹配大幅减少误判。

可用操作：
- **禁言** — 临时限制发言（可设时长）
- **踢出** — 移出群组（可重新加入）
- **封禁** — 永久移出群组

## 渠道

**渠道** 是 Agent 连接的消息平台。目前支持：

- **Telegram** — 完整支持，包括群组管理、@提及控制、多语言回退
- **Discord** — 即将推出
- **Slack** — 即将推出

## 回退语言

当 Agent 无法检测用户语言时，使用配置的回退语言。支持英语、中文、韩语、土耳其语、印尼语、俄语和越南语。默认为 **跟随用户**（自动检测）。
