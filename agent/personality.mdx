# Personality Configuration

Personality defines the tone, style, and behavior of your AI agent. You can choose a preset or write a fully custom personality.

## Preset Personalities

Select from four built-in profiles:

| Preset | Description |
|--------|-------------|
| **Friendly** | Casual and approachable. Uses emoji, conversational tone, and warm language. |
| **Professional** | Formal and structured. Business-appropriate responses with clear formatting. |
| **Creative** | Imaginative and playful. Uses metaphors, humor, and creative expression. |
| **Concise** | Brief and direct. Minimal words, maximum clarity. No unnecessary filler. |

Click any preset card to apply it. The chat preview on the right updates immediately so you can see how the agent will respond.

## Custom Personality

Select **Custom** to write your own personality description. This is a free-form text field where you describe:

- How the agent should speak (tone, formality, language style)
- What topics to focus on or avoid
- Any specific instructions or constraints
- Character traits or persona details

**Example:**
```
You are a helpful Web3 community manager. Respond in a friendly but knowledgeable tone. 
Always link to official documentation when answering technical questions. 
Use simple language to explain complex blockchain concepts.
Never give financial advice.
```

<Tip>Switching between presets preserves your custom text. You can experiment with presets and switch back to Custom without losing your work.</Tip>

## Saving Changes

After making changes, click **Save** at the bottom of the page. The button only appears when you have unsaved changes.

<Note>Changes take effect immediately after saving — no restart required.</Note>
